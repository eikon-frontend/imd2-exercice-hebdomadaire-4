# Exercice hebdomadaire 4

## Installation

1. Cloner le repository
2. Ouvrir le dossier avec Visual Studio Code
3. Ouvrir une fenêtre de terminal dans Visual Studio Code et entrer la commande suivante :

```bash
npm install
```

## Travail

Depuis le terminal, entrer la commande suivante :

```bash
npm run dev
```

## Consigne

1. Ouvrir [la maquette Figma](https://www.figma.com/design/X40O5X4cf3eGeSqmKWn03j/imd2-exercice-hebdomadaire-4?node-id=0-1&t=hWZGJoAcXQ4qjuGp-1) dans l'équipe de votre année
2. Intégrer la maquette aussi précisément que possible, en reprenant les propriétés de Figma
3. Faire un commit, puis un push
